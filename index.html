My Projects

Face Recognition Attendance System

A Face Recognition Attendance System is a smart application that automatically marks attendance by identifying individuals through their facial features.

...

Instead of using manual registers or ID cards, the system captures a person’s face through a camera and compares it with a stored database of registered users. If a match is found, attendance is marked instantly, making the process fast and efficient.

The system is usually built using Python with libraries like OpenCV and face recognition models to detect and encode faces. It first stores multiple images of each user during registration, then uses real-time video processing to recognize faces during attendance sessions. The data is then saved into a database such as SQLite or MySQL, including time, date, and user details.

This project is highly useful in schools, colleges, and workplaces because it reduces proxy attendance and saves time. It also improves accuracy and security while making the system fully automated. With AI integration, it can further be enhanced with features like real-time reports, dashboards, and mobile access.

Real Estate Plot Price Prediction

A Real Estate Plot Price Prediction system is a machine learning-based application that estimates the future or current price of land plots based on different features.

...

These features may include location, plot size, area type, nearby facilities, market trends, and historical property prices. The goal is to help buyers, sellers, and investors make better and more informed decisions.

The system is usually built using Python and data science libraries such as Pandas, NumPy, and Scikit-learn. It uses regression algorithms like Linear Regression, Decision Trees, or Random Forest to learn patterns from past real estate data. After training, the model can take user inputs like plot area and location and predict an estimated price with reasonable accuracy.

This project is very useful in the real estate industry because it reduces uncertainty and helps in fair pricing of properties. It can also be integrated into web or mobile applications (like Streamlit or Flask apps) where users can instantly check predicted prices. With advanced AI, it can further improve by using location maps, market trends, and live property data.

Fire Detection

This project is a Fire Detection Alarm System developed using Python, OpenCV, and computer vision techniques.

...

This project is a Fire Detection Alarm System developed using Python and OpenCV. It uses real-time video input from a webcam to monitor the environment and detect fire based on color recognition techniques.

The core functionality relies on detecting fire-like colors such as red, orange, and yellow in the HSV color space. By analyzing pixel intensity and filtering noise, the system can distinguish fire regions from the background and highlight them on the video feed.

The system processes each video frame by converting it into HSV color space and identifying fire-like colors such as red, orange, and yellow. When the detected fire pixels exceed a certain threshold, the system highlights the area and confirms the presence of fire.

Once fire is detected, an alarm sound is triggered instantly to alert the user. This project demonstrates skills in computer vision, real-time processing, and building practical safety applications, with potential for future enhancement using AI-based detection methods.
