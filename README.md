# Portfolio Website

## Overview
This is a personal portfolio website showcasing projects, skills, and contact information. Built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Project showcase with videos and images
- Profile section

## Files
- `index.html`: Main homepage
- `all-projects.html`: Projects page
- `style.css`, `all-projects.css`: Styling
- `script.js`, `all-projects.js`: JavaScript functionality
- `Images/`: Profile and section images
- `Project_Videos/`: Project demo videos

## Usage
1. Open `index.html` in a web browser.
2. No additional setup required.
